Puns As A Service

Hello! Welcome to PaaS, or Puns as a Service. This was created by Thomas Kercheval, Chris Johnston, and Lizzy Presland for the UWB-ACM. Check out the source code here.

Puns API

This Flask application serves only the finest Computer Science-related puns using a REST API.
  • Try the /random endpoint to get a random pun!
  • Try the /pun/id endpoint to get a pun by it's ID.
